# Difference Checklist

현재 복제본 기준으로 확인한 차이점과 상태입니다.

- 시각 구조, 섹션 순서, 헤더 오버레이, 히어로 배치, 주요 다크/라이트 전환, 모바일 메뉴 배치는 원본과 거의 동일하게 맞춰져 있습니다.
- 메뉴 진입 페이지인 `ABOUT`, `MARKETING(/18)`, `CONTACT`도 함께 미러링했고, 로컬 메뉴 클릭 시 동일한 경로로 이동하도록 맞췄습니다.
- `CONTACT` 캘린더에서 필요로 하는 일정/권한/색상 AJAX 응답도 로컬 fixture로 재현해 초기 오류창이 뜨지 않도록 맞췄습니다.
- 원본 웹폰트와 vendor CDN import를 그대로 사용하고 있어 별도 폰트 대체는 하지 않았습니다. 현재 확인 기준에서 폰트 차이는 없습니다.
- 원본의 분석/로그용 스크립트 중 `datadog`와 `brandscope`는 시각 재현에 불필요해 제거했습니다.
- `npm start`에 포함된 Node 서버는 몇몇 `imweb` 백엔드 호출에 대해 빈 응답을 반환합니다. 이 부분은 시각 복제에는 영향이 없지만, 순수 정적 호스팅만 할 경우 동일한 스텁은 별도 서버 설정이 필요합니다.
- 뉴스레터 구독, 회원 관련 기능, 검색, 일부 로그/토큰 API는 현재 복제본에서 시각/프론트 동작 중심으로 유지되어 있으며, 실제 서비스용 백엔드까지 독립 운영하려면 별도 연동이 필요합니다.
